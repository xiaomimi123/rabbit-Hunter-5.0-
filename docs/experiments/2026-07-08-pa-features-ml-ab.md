# 实验归档：价格行为学特征 → ML 模型 A/B

**日期**: 2026-07-08
**结论**: **不晋升**（真实但微弱：Δ +0.003 < 门槛 +0.01），列入数据积累后复审清单

## 背景

操盘员提议"让 AI 学习价格行为学配合现有策略"。盘点发现：
- PA 规则策略已存在（v0.3.1）且已被数据禁用（1620 笔 / 34% WR / -$1250）
- 但 ML 模型的 18 特征全是指标类，**从未包含形态特征** —— 真空白

## 实验（Kronos 同款协议）

2996 笔交易（统一无 NaN 样本 2373 笔），同切分同超参 LightGBM：

| 组 | 特征 | test AUC | Δ |
|---|---|---|---|
| 控制 | 18 指标 | 0.5170 | — |
| T1 | +7 形态旗标（吞没×2/pinbar/inside/doji/marubozu/BOS） | 0.5221 | **+0.0050** |
| T2 | T1 + 摆动距离×2 + 结构 one-hot×2 | 0.5143 | -0.0027 ❌ |

T1 分割稳健性探针（5 个切分点，特征集不动）：
+0.0012 / +0.0001 / +0.0050 / +0.0052 / +0.0031 → **5/5 正向，均值 +0.0029**

## 解读

1. 形态旗标信号**真实**（方向 5/5 一致，非 Kronos 式噪音）但**微弱**（门槛的 1/3）
2. 派生特征（摆动距离/结构 one-hot）负贡献 —— 结构信息与 EMA 排列特征冗余
3. 机制解释：extreme momentum gate 把入场压缩到极端时刻，形态在此处
   稀疏（0.5-8.7% 非零）且信息已被 RSI/z-score 表达。事件研究中形态
   确认的高价值（+0.3~0.9R）属于回调入场体系 —— 本系统不采用
4. PA 思想在本系统的正确形态已被 v0.1.10 证明：**作为门控**
   （structure_regime 禁震荡多 → Sharpe +53%），而非入场信号或 ML 特征

## 复审条件

shadow 积累使训练集扩大（例如 +50% 交易）后重跑本 A/B —— 微弱信号
在更大样本上可能过门槛，估计方差也更小。
